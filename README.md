# RAG UI sample

これは RAG の UI サンプルです。  
React×Typescript での実装となります。

## Backend

バックエンドは API となっており、下記 URL を参照してください  
https://github.com/takuya-fukuda/rag

### 起動方法

npm start

####　フォルダ説明

pages: アプリケーションルート先のファイル
components/api: アプリケーション内で呼ばれる API の部品
components/layout:　レイアウト用の部品が入っている
hooks: API の関数格納先
